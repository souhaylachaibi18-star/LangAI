# LangAI 🚀
An intelligent and interactive application designed to bridge the gap and facilitate learning Chinese Mandarin for Arabic speakers, built entirely using Python.

## 🌟 Key Features:
- **Multilingual UI:** A smooth, intuitive, and user-friendly interface that supports multiple language options.
- **Generative AI Integration:** Powered by advanced AI to deliver personalized, contextual grammar explanations and real-time sentence analysis.
- **Audio Pronunciation:** High-quality, clear native audio playback for vocabulary words to ensure accurate pronunciation.
- **Interactive Quiz Subsystem:** A built-in testing feature to challenge and evaluate vocabulary retention and track learner progress.

## 🛠️ Tech Stack:
- **Frontend & UI:** Streamlit
- **AI Core:** Pollinations AI API (Generative AI)
- **Data Management:** Pandas
- **Audio & TTS:** gTTS (Google Text-to-Speech)
